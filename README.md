# Network UI

Python-утилита для временной блокировки сети у конкретного процесса через Windows Firewall.

## Запуск

Из корня репозитория:

```powershell
python .\scripts\network-ui\process_network_ui.py
```

Или из самой папки:

```powershell
python .\process_network_ui.py
```

## Что умеет

- показать запущенные процессы с путями к `.exe`
- отфильтровать список
- выбрать произвольный `.exe`
- блокировать сеть
- блокировать сеть на `N` секунд
- вручную разблокировать сеть
- проверить статус блокировки

## Важно

- запускать от имени администратора
- используется `Windows Firewall`
- если Python уже запущен не от администратора, приложение попытается перезапустить себя с elevation
